What it does

A channel-agnostic pipeline: describe a channel once, then produce validated videos on demand.

🎬 One topic in, a video out

Research → script → direction → voice → keyframes → clips → assembly → captioned 1080×1920 cut, with validation gates.

🧩 Channel-agnostic

A channel is a folder: premise, persona, beat structure, look. Science explainers, history, cooking — same engine.

💸 $0 creative cost

Script, direction, judging and SEO run on your Claude subscription. You pay only for AI image/video generation.

🖥️ UI or CLI

A local browser control center or a headless CLI — they share artifacts and resume each other freely.

🔁 Resumable & versioned

Rerun after any failure; cached steps skip. Nothing is deleted — superseded artifacts are archived and restorable.

🔌 Extend via registries

A new research source, generation backend, or TTS engine is one registry entry — not an if/elif chain.

Quick start

Python 3.9+, ffmpeg, and one-time logins for the two CLIs Kalinga drives — no API keys required.

# 1. Python deps
python3 -m pip install -r requirements.txt

# 2. ffmpeg (assembly; ffmpeg-full for burned-in karaoke captions)
brew install ffmpeg

# 3. The two CLIs (one-time logins, no API keys)
curl -fsSL https://claude.ai/install.sh | sh
curl -fsSL https://raw.githubusercontent.com/higgsfield-ai/cli/main/install.sh | sh
higgsfield auth login

# 4. Guided first-run setup
python3 kalinga.py init

# 5. Open the control center (both demo channels ship a finished video)
python3 kalinga.py ui

Two example channels ship with a finished demo video already in the repo — explore before you spend a credit.

Inspect and edit every stage

Voice, keyframes, and assembly are all editable in the browser before and after render.
